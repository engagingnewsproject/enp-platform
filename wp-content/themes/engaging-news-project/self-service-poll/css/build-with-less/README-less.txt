1. Add less code to main.less
2. Make sure it only points to other less files
3. Install less (http://lesscss.org/)
4. Run the following
lessc wp-content/themes/engaging-news-project/self-service-poll/css/build-with-less/main.less wp-content/themes/engaging-news-project/self-service-poll/css/build-with-less/main-less.css 
5. Then rename main-less.css, as it will have your newly created code
